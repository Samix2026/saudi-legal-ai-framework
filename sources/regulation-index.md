# Regulation Index / سجل الأنظمة الموحَّد

**هذا الملف هو المصدر الرسمي الوحيد لصيغة الاستشهاد بالأنظمة السعودية في هذا الإطار.**
**This file is the single authoritative citation source for Saudi regulations across this framework.**

عند الاستشهاد بأي نظام في أي ملف آخر بالمشروع، استخدم **صيغة الاستشهاد الموحَّدة** المذكورة هنا بالضبط.
When citing any regulation in any other project file, use the **Common Citation Format** listed here verbatim.

---

## كيفية الاستخدام / How to Use

- **في ملفات `skills/` و`prompts/` و`examples/`:** استخدم عمود "صيغة الاستشهاد" مباشرةً.
- **في عمود `related_regulation` بالـ dataset:** استخدم نفس الصيغة — أضف رقم المادة بعدها إذا احتجت: `... Art. 84`
- **عند صدور تعديل على نظام:** حدّث هذا الملف أولًا، ثم ابحث عن جميع الاستشهادات في المشروع وعدّلها.

- **In `skills/`, `prompts/`, `examples/` files:** use the Citation Format column directly.
- **In the dataset `related_regulation` column:** use the same format — append the article number if needed: `... Art. 84`
- **When a regulation is amended:** update this file first, then search the project for all citations and update them.

## إضافة نظام جديد / Adding a New Regulation

لإضافة نظام جديد لهذا السجل:
1. أضف قسمًا بالترقيم التالي في القائمة أعلاه
2. أضف سطرًا في جدول المرجع السريع
3. افتح Pull Request مع رابط المصدر الرسمي من boe.gov.sa أو uqn.gov.sa

To add a new regulation to this registry:
1. Add a numbered section to the list above
2. Add a row to the quick reference table
3. Open a Pull Request with a link to the official source from boe.gov.sa or uqn.gov.sa

## مصادر البيانات المفتوحة الرسمية / Official Open Data Sources

> هذا القسم يُدرج مصادر البيانات المفتوحة الحكومية ذات الصلة بالمنظومة القانونية السعودية.
> هذه المصادر **مكمِّلة** — تُفيد في الإحصاء والأنماط وإشارات الممارسة القضائية، **ولا تُعوِّض** النصوص التشريعية الرسمية.
>
> This section lists official government open data sources relevant to the Saudi legal system.
> These sources are **supplementary** — useful for statistics, patterns, and court practice signals; they do **not** replace official legislative texts.
