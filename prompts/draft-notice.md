# Prompt Template: Draft Legal Notice

> This template works with Claude, ChatGPT, and Gemini

---

## ⚠️ تحذير / Warning

> **تحذير:** الإشعارات القانونية المُنتجة بالذكاء الاصطناعي هي مسودات أولية للمراجعة فقط. لا تُرسَل في أي معاملة رسمية دون مراجعة وتوقيع محامٍ مرخّص في المملكة العربية السعودية.
>
> **Warning:** AI-generated legal notices are preliminary drafts for review only. Do not send in any official transaction without review and sign-off from a licensed attorney in the Kingdom of Saudi Arabia.

## القالب العام / General Template

```
أنت متخصص في الصياغة القانونية السعودية. أنشئ إشعارًا قانونيًا رسميًا وفق المعايير الآتية:

المتطلبات:
1. الصياغة باللغة العربية أولًا (اللغة القانونية الرسمية في المملكة)، ثم الإنجليزية إذا طُلبت
2. هيكل رسمي يشمل: التاريخ بالهجري والميلادي، الطرفين، موضوع الإشعار، المطالبة، المهلة، التبعات
3. أسلوب قانوني رصين دون تهديد أو إساءة
4. لا تتضمن أرقام قضائية أو مراجع حقيقية لم تُزوَّد بها
5. أضف ملاحظة في النهاية: "هذه مسودة أولية — يجب مراجعتها من محامٍ مرخّص قبل الإرسال"

معلومات الإشعار:
- المُرسِل (الجهة المُصدِرة): [اسم الجهة + سجلها التجاري إن كانت شركة]
- المُرسَل إليه: [اسم المستلم + عنوانه]
- نوع الإشعار: [مطالبة / إنهاء / إنذار / غيره]
- الموضوع: [وصف موجز للخلاف أو الحادثة]
- المطالبة المحددة: [ما تطلبه بالضبط]
- المهلة الممنوحة: [عدد الأيام]
- التبعات في حال عدم الامتثال: [ما ستتخذه من خطوات]
- مستند الاستحقاق: [رقم العقد / الفاتورة / التاريخ]
```

## قوالب متخصصة / Specialized Templates

### إشعار مطالبة بدين / Debt Collection Notice

```
أنت متخصص في الصياغة القانونية التجارية السعودية.

اكتب إشعار مطالبة بدين تجاري وفق القانون السعودي يتضمن:
- رأس الصفحة: اسم الجهة المُرسِلة، تاريخ الإشعار (هجري وميلادي)، رقم الإشعار
- عنوان: "إشعار مطالبة"
- الفقرة الأولى: تحديد العلاقة التعاقدية (اسم العقد / الفاتورة / تاريخ الاستحقاق)
- الفقرة الثانية: تحديد المبلغ المستحق (رقمًا وكتابةً) ومدة التأخر
- الفقرة الثالثة: المهلة الممنوحة للسداد (يُوصى بـ 14 يومًا)
- الفقرة الرابعة: التبعات عند عدم السداد (اللجوء للقضاء التجاري)
- الخاتمة: التوقيع المخوَّل + الختم

المعلومات المطلوبة:
- الدائن: [CREDITOR_NAME]
- المدين: [DEBTOR_NAME]
- المبلغ: [AMOUNT] ريال سعودي
- المستند: [CONTRACT_OR_INVOICE_REF]
- تاريخ الاستحقاق الأصلي: [DUE_DATE]
- المهلة الجديدة: [GRACE_PERIOD] يومًا
```

### إشعار إنهاء عقد / Contract Termination Notice

```
أنت متخصص في الصياغة القانونية السعودية.

اكتب إشعار إنهاء عقد وفق نظام المعاملات المدنية السعودي يتضمن:
- تحديد العقد المُنهى (رقمه وتاريخه وأطرافه)
- الأساس القانوني أو التعاقدي للإنهاء
- تاريخ نفاذ الإنهاء
- الالتزامات المتبقية على كل طرف بعد الإنهاء
- آلية تسوية المستحقات المتبادلة
- إشعار باسترداد أي ممتلكات أو وثائق

المعلومات المطلوبة:
- الطرف المُنهي: [TERMINATING_PARTY]
- الطرف الآخر: [OTHER_PARTY]
- رقم / موضوع العقد: [CONTRACT_REF]
- سبب الإنهاء: [REASON]
- تاريخ نفاذ الإنهاء: [EFFECTIVE_DATE]
- الالتزامات المتبقية: [REMAINING_OBLIGATIONS]
```

### إشعار إنهاء عقد عمل / Employment Termination Notice

```
أنت متخصص في نظام العمل السعودي (م/51 لعام 1426هـ).

اكتب إشعار إنهاء عقد عمل يراعي متطلبات نظام العمل السعودي:
- تحديد نوع العقد (محدد / غير محدد المدة) لأن ذلك يؤثر على الإجراء
- تحديد سبب الإنهاء (إذا كان بعذر مشروع: الإشارة إلى الفقرة المنطبقة من المادة 80)
- مدة الإشعار المسبق (60 يومًا للشهريين — أو الإشارة إلى الاتفاق المخالف للحد)
- تحديد التاريخ الأخير للعمل
- تذكير باستحقاقات نهاية الخدمة وموعد صرفها
- تذكير بتسليم مستندات الموظف (شهادة الخبرة، مطالبات GOSI)

المعلومات المطلوبة:
- صاحب العمل: [EMPLOYER_NAME]
- الموظف: [EMPLOYEE_NAME]
- المسمى الوظيفي: [JOB_TITLE]
- تاريخ بدء الخدمة: [START_DATE]
- سبب الإنهاء: [REASON]
- التاريخ الأخير للعمل: [LAST_WORKING_DAY]
```

## متطلبات الشكل الرسمي / Formal Requirements Checklist

قبل إرسال أي إشعار قانوني، تأكد من:

- [ ] التاريخ مذكور بالتقويمين الهجري والميلادي
- [ ] بيانات الطرفين كاملة (الاسم الكامل، السجل التجاري إن وُجد، العنوان)
- [ ] موضوع الإشعار محدد في سطر واضح
- [ ] المطالبة محددة وقابلة للقياس
- [ ] المهلة الزمنية واضحة وتبدأ من تاريخ التسلّم لا التحرير
- [ ] التوقيع صادر ممن يملك صلاحية التوقيع (مع إرفاق التفويض إن احتيج)
- [ ] الختم التجاري (للشركات)
- [ ] طريقة التوصيل تُنشئ إثباتًا (بريد مسجل، بريد رسمي، ناجز)
